## 🌟 Project Overview

An interactive **command-line quiz game** built using Python 🐍 that challenges users with multiple-choice questions across different difficulty levels.

This project demonstrates core programming concepts in a fun and engaging way 🎮

---

## 🚀 Features

✨ 3 Difficulty Levels (Easy 😄 → Medium 😊 → Hard 🫡)
🎯 Real-time Score Tracking
🔁 Replay Option
⚡ Instant Feedback (Correct / Wrong answers)
🎨 Clean CLI Interface
🧠 Python-based logical questions

---

## 🧠 Concepts Used

* 🔁 Loops (`while`)
* 🔀 Conditional Statements (`if-else`)
* ⌨️ User Input Handling
* 📊 Variables & Score Tracking
* 🎯 Logical Thinking

---

## 🎮 How to Play

1. Run the program ▶️
2. Enter your name 👤
3. Read the rules 📜
4. Answer questions from all 3 levels
5. Get 1 point for each correct answer ⭐
6. View your final score 🎯
7. Choose to play again 🔁
